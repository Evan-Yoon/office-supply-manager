# 🏢 사내 비품 재고 관리 및 자동 발주 시스템 (Office Supply Manager)

## 📌 프로젝트 기획 의도
경영지원(HR/GA) 실무 중 매주 반복되는 **비품 재고 파악 및 발주 품의서 작성 시간을 획기적으로 단축**하기 위해 기획한 파이썬 자동화 스크립트입니다. 

스타트업 및 신설 조직 초기 세팅 시 발생하는 방대한 총무 행정 업무를 시스템으로 자동화하고, 이를 통해 **절약된 리소스를 수평적 조직문화 구축, 핵심 인재 채용 등 고부가가치 HR 업무에 집중**하는 것을 목표로 합니다.

## ✨ 주요 기능
1. **재고 자동 분석:** `inventory.csv` 데이터를 읽어 품목별 현재 재고량과 '최소 유지 수량'을 비교 분석합니다.
2. **소요 예산 자동 계산:** 발주가 필요한 품목의 수량을 파악하고, 단가를 곱해 총 예상 결제 금액을 산출합니다.
3. **발주서 자동 생성:** 당일 날짜가 기록된 실무용 텍스트 파일(`비품발주요청서_YYYY-MM-DD.txt`)을 즉시 생성하여 담당자의 수작업을 없앱니다.

## 🛠 사용 기술 및 라이브러리
- **Language:** Python 3.x
- **Libraries:** `csv`, `datetime`, `os` (별도의 설치가 필요 없는 내장 라이브러리 활용으로 범용성 확보)

## 🚀 기대 효과
- **Human Error 방지:** 수기 계산으로 인한 수량 및 금액 오기입을 원천 차단합니다.
- **업무 효율화:** 비품 파악부터 기안 작성까지 걸리던 시간을 단 1초로 단축시켜, 담당자의 업무 생산성을 극대화합니다.
